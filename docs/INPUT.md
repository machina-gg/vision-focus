# 要件ヒアリングシート

## 1. 何を作りたいか？

**サービス名：VisionFocus（ヴィジョン・フォーカス）**
「視覚を支配し、理想を現実にする」ためのChrome拡張機能。
単なるサイトブロッカーではなく、 **「現状の可視化（分析）」「誘惑の遮断（ブロック）」「目標の再認識（モチベーション）」** の3段階で、ユーザーの生産性を最大化するセルフコントロール・システム。

## 2. なぜ作りたいか？

- **個人的な動機:** 優秀なライバルに対して、仕事の成果と個人開発での収益で圧倒的に勝利するため。もう二度と悔しい思いをしないため。自分の理想を実現するため。
- **市場の課題:** 既存のブロッカーは「禁止」するだけで動機付けが弱く、また最新のツールはAI多用により動作が重く、かつ高価である。
- **解決策:** 非AIによる爆速な動作と、月額1ドルという低価格、そして「なぜ今それをしてはいけないのか」を視覚的に訴えかけるページ表示により、ユーザーを強力に目標へ引き戻す。

## 3. 誰が使うか？

- **コアターゲット:** 圧倒的な成果を出して人生を変えたい、あるいはライバルを追い抜きたいと考えている若者。まだまだ夢を追いかけ、頑張りたい中堅層。
- **属性:** 無駄な時間を削って個人開発や学習に充てたいが、ついついSNSやYouTubeを見てしまう意志の弱い（しかし向上心のある）層。

## 4. 必須機能は何か？

- **スマート・サイトブロック:** 登録したドメインへのアクセスを遮断。解除に手間（タイピング練習等）をかける設定も可能。
- **ビジョン・ダッシュボード（新タブ/ブロック時）:** - 自分の目標、ライバルへの対抗心、理想の生活などを自由にカスタマイズして表示。
- 背景画像（Unsplash連携や自前アップロード）やフォントのカスタマイズ機能。

- **タイム・アナリティクス（非AI・ローカル集計）:** - どのサイトにどれだけ滞在したかを1秒単位で記録。
- 「浪費時間」と「投資時間（学習等）」を可視化するグラフ表示。

- **プライバシー保護:** 全ての閲覧データはサーバーに送信せず、ローカルストレージで完結。

## 5. マネタイズ

**フリーミアムモデル（月額1ドル / 年額10ドル）**

- **無料版:** 基本的なサイトブロック（数制限あり）、当日の分析レポート、デフォルトの目標表示。
- **有料版（1ドル）:** 無制限のブロック、全期間の分析グラフ、背景画像のアップロード、外部サービス（GitHub等）との連携表示、強制解除不可の「ハードモード」。

## 6. その他

- **技術的要望:** - AIを使わず軽量なロジックで実装し、APIコストをゼロにする。
- Manifest V3に完全準拠し、プライバシー第一（Local-First）を掲げる。

- **デザインイメージ:** - 無駄がなく、プロフェッショナルな「計器（コンソール）」のようなUI。
- ブラウザを立ち上げた瞬間に、自分の目標が鮮明に飛び込んでくるUX。
