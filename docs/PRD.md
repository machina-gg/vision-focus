# VisionFocus

## 1. 概要

VisionFocusは、「視覚を支配し、理想を現実にする」ためのChrome拡張機能。単なるサイトブロッカーではなく、「現状の可視化（分析）」「誘惑の遮断（ブロック）」「目標の再認識（モチベーション）」の3段階で、ユーザーの生産性を最大化するセルフコントロール・システム。

## 2. 背景・課題

- 優秀なライバルに対して、仕事の成果と個人開発での収益で圧倒的に勝利したい

### 市場の課題

- 既存のブロッカーは「禁止」するだけで動機付けが弱い

- 「なぜブロックされているのか」を思い出させる仕組みがない

### 解決策

- 「なぜ今それをしてはいけないのか」を視覚的に訴えかけるページ表示
- 全データをローカルに保持するプライバシーファースト設計

## 5. 非機能要件

### 対応環境

- **ブラウザ**: Google Chrome（最新版）
- **Chrome拡張機能**: Manifest V3準拠
- **レスポンシブ**: デスクトップのみ（Chrome拡張のため）

### パフォーマンス

- 拡張機能のロード時間: 100ms以下
- ブロックページ表示: 即時
- 分析データの更新: リアルタイム（1秒単位）
- メモリ使用量: 50MB以下

### セキュリティ・プライバシー

- 全データをローカルストレージに保存
- 外部サーバーへのデータ送信なし
- 最小限の権限要求
- chrome.storage.local APIのみ使用

### 保守性

- TypeScriptによる型安全性
- ESLint/Prettierによるコード品質維持
- Vitestによる単体テスト

### 多言語対応

- **対応言語（MVP）**: 英語（デフォルト）、日本語
- **言語判定**: ブラウザの言語設定による自動判定（手動切替不要）
- **実装方式**: chrome.i18n API
- **対象範囲**:
  - 拡張機能内のすべてのUI
  - Chrome Web Storeのストアリスティング（説明文・スクリーンショット）
- **将来対応**: スペイン語、中国語（簡体字）、韓国語等

- ブロックサイト: 無制限

- ダッシュボードスタイル: 最大5件

### Free ダウングレード時の動作

有料版から無料版にダウングレードした場合、既存データは削除されず、再アップグレード時に復元される。

| 機能                   | 動作                               | データ |
| ---------------------- | ---------------------------------- | ------ |
| カスタム背景           | デフォルト背景にフォールバック     | 保持   |
| 2件目以降のスタイル    | 選択・適用不可（🔒表示）           | 保持   |
| スケジュールのスタイル | ロックされたスタイルは適用されない | 保持   |
| 超過分のブロックリスト | 引き続きブロック有効               | 保持   |

## 8. リスクと対策

| リスク                        | 影響度 | 対策                                     |
| ----------------------------- | ------ | ---------------------------------------- |
| Manifest V3への完全準拠が困難 | 高     | 開発初期から準拠を徹底                   |
| 無料競合との差別化失敗        | 高     | ビジョン・ダッシュボードの独自価値を強調 |
| Chrome Web Store審査落ち      | 中     | ポリシーを事前に熟読、段階的リリース     |
| 有料化への心理的抵抗          | 中     | 無料版でも価値を体験できる設計           |
| プライバシーポリシー対応      | 低     | サーバー通信なしで対応不要項目を最小化   |
