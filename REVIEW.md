# Review Guidelines

Claude Code Action がこのファイルを参照してレビューを実施する。
リポ固有のルール（技術スタック、コーディング規約等）は CLAUDE.md を参照すること。

## コード品質

- 命名が明確で意図が伝わるか
- 不要な複雑さ・重複がないか
- SOLID 原則に沿っているか
- エラーハンドリングが適切か

## セキュリティ

- ユーザー入力のサニタイズ・バリデーション
- インジェクション（SQL / XSS / コマンド）の防止
- シークレット・クレデンシャルのハードコード
- 認証・認可の実装

## パフォーマンス

- N+1 クエリ等の非効率なデータ取得
- 不要な再レンダリング・再計算
- 適切なキャッシュ・メモ化

## アーキテクチャ

- 適切な責務分離（関心の分離）
- 変更の影響範囲が適切か
- 既存パターンとの一貫性
